# Suggestion::class
    * **Tabla:** `suggestions`
    * [Diccionario](/docs/base-de-datos#sugerencias)

        Representa las sugerencias que un usuario puede realizar.

**Metodos:**

    * `user()`: Devuelve el [Usuario](/docs/base-de-datos#equipos)(*proveedor*) que realizó la sugerencia.
