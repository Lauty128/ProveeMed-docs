# Option::class
    * **Tabla:** `option`
    * [Diccionario](/docs/base-de-datos#marcas)

**Metodos:**

    * `user_types()`: Devuelve listado de los [Tipos de usuarios](/docs/base-de-datos#opciones) con acceso a esta opción.
